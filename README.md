## Installation
Clone the repository:
```bash
git clone https://github.com/Eslam-Radwan/E-Learning-Website.git
```


### Navigate to the project directory:

```bash
cd E-Learning-Website
```

### Install dependencies:
```bash
npm install


```
### Start the development server:
```bash
npm run dev
```
